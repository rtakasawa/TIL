# sql

### select文の実行順
```
FROM句
↓
JOIN句
↓
WHERE句
↓
GROUP BY句
↓
HAVING句
↓
SELECT句
↓
ORDER BY句
↓
LIMIT句
```
※引用先：https://qiita.com/k_0120/items/a27ea1fc3b9bddc77fa1

#### 例
```
shop=# select shohin_bunrui,count(hanbai_tanka)
from shohin
group by shohin_bunrui;
 shohin_bunrui | count 
---------------+-------
 キッチン用品  |     4
 衣服          |     2
 事務用品      |     2
(3 rows)
```
